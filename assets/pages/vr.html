Virtual Reality

We are working with all the latest hardware: Oculus Rift DK2, Samsung Gear VR, Kinect, Leap motion, Google Cardboard, etc.

1. We are using WebVR, this is an experimental Javascript API that provides access to Virtual Reality devices, such as the Oculus Rift or Google Cardboard, in your browser.

2. You can see some examples with WebVR, API with these samples:

  • WEB PANORAMIC TOUR - bkeller using: krpano Plugins / krpano Panorama Viewer

  • INTERACTIVE 3DMODEL on WEBSITE - Renault Escape using: WebGL / WebVR technologies / threejs / 3D Content Creation

  • CGI STEREOSCOPIC VR webtour - Arc Media using: stereoscopic cube maps generated with 3DsMax

  • WebVR Panoramic Video - Krpano using: krpano Plugins / krpano Panorama Viewer (Flash and HTML5) viewer

360 Видео Продакшн 360 видео перемещает зрителя внутрь фильма, позволяя видеть все вокруг себя, по-настоящему погружает в контент. Мы используем различные 360 камеры и профессиональные технологии сшивания для реализации проектов от прямых трансляций до кинематографических ВР фильмов.
